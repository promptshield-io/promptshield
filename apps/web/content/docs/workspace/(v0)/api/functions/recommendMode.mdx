# Function: recommendMode()

> **recommendMode**(`fileCount`, `cacheSplitThreshold?`): `"single"` \| `"split"`

Defined in: [cache.ts:173](https://github.com/promptshield-io/promptshield/blob/main/packages/workspace/src/cache.ts#L173)

Recommends optimal cache mode based on repository size.

Heuristic:
- Small/medium repositories → `single`
- Large repositories → `split`

This function is deterministic and side-effect free.
It does not perform I/O.

## Parameters

### fileCount

`number`

### cacheSplitThreshold?

`number` = `CACHE_SPLIT_THRESHOLD`

## Returns

`"single"` \| `"split"`
