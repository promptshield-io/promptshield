# Security Policy

## Version Support

We provide support for the latest minor version. Pull requests aimed at fixing security vulnerabilities in the version immediately preceding the latest will be considered. Support for versions prior to this relies entirely on community-driven pull requests.

We provide support for the **latest minor version** of each package.

## Patching Long-Term Support (LTS) Versions

If you rely on a previous minor version of any of the packages in this repo and need to address security issues, kindly submit a pull request to the relevant branch.

Ensure to add appropriate changeset using @changeset/cli.

Note: We only accept pull requests addressing security vulnerabilities. Additional functionalities and bug fixes for older versions are beyond the scope.

## Reporting Vulnerabilities

Kindly report vulnerabilities [here](https://github.com/promptshield-io/promptshield/security/advisories/new).

Refrain from sharing vulnerabilities publicly or privately via any other channel until a fix is published.
