# Enumeration: ThreatCategory

Defined in: [types.ts:38](https://github.com/promptshield-io/promptshield/blob/main/packages/core/src/types.ts#L38)

Categories of threats detected by PromptShield.

Categories describe the *type of attack vector* rather than
the specific implementation detail.

## Enumeration Members

### Homoglyph

> **Homoglyph**: `"HOMOGLYPH"`

Defined in: [types.ts:55](https://github.com/promptshield-io/promptshield/blob/main/packages/core/src/types.ts#L55)

Mixed-script characters that visually resemble others.

Example:
Cyrillic "Ð°" vs Latin "a".

### Invisible

> **Invisible**: `"INVISIBLE_CHAR"`

Defined in: [types.ts:47](https://github.com/promptshield-io/promptshield/blob/main/packages/core/src/types.ts#L47)

Invisible Unicode characters such as:
- Zero Width Space (ZWSP)
- Zero Width Joiner (ZWJ)
- BIDI markers

These are commonly used for prompt injection smuggling.
