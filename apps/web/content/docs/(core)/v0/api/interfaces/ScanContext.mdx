# Interface: ScanContext

Defined in: [types.ts:185](https://github.com/promptshield-io/promptshield/blob/main/packages/core/src/types.ts#L185)

Execution context for scanning text fragments.

Used when scanning partial content extracted from a larger source,
such as:
- markdown code blocks
- AST nodes
- editor buffers
- diff hunks
